## 📝 Description

Leetcode-frontend is a modern, responsive web application built with React, designed to emulate the experience of platforms like LeetCode. It provides a clean and interactive interface where users can browse coding problems, write and execute code, and track their progress — making it ideal for interview preparation and competitive programming practice.

## 🌐 Live Demo

🔗 Frontend: https://leetcode-frontend-sage.vercel.app/
🔗 Backend: https://leetcode-backend-smd2.onrender.com

> ⚠️ Notes:
>
> * If the site doesn’t load on college WiFi, try using your own internet connection.

## ✨ Features

* 🧠 Solve coding problems with real-time execution
* ⚡ Monaco-powered code editor (VS Code-like experience)
* 📊 Submission history tracking
* 🎯 Filter and search problems
* 🔐 Authentication (Login / Signup)
* 🛠️ Admin panel for managing problems
* 🎥 Video solution upload & viewing
* 🤖 AI Chat support
* 🌙 Dark mode support
* 📱 Fully responsive design

## 🛠️ Development Setup

### Prerequisites

* Node.js (v18+ recommended)

### Installation

git clone https://github.com/Abhinav22choubey/Leetcode-frontend.git
cd Leetcode-frontend
npm install
npm run dev

---

## 🔐 Environment Variables

Create a `.env` file and add:

VITE_BACKEND_URL=https://leetcode-backend-smd2.onrender.com
